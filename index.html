Introduction to JavaScript

JavaScript is a powerful, high-level programming language used for creating dynamic and interactive websites. It's essential for both client-side and server-side development. JavaScript is used in frameworks like React, Angular, and Node.js.

1.1 History of JavaScript

JavaScript was created in 1995 by Brendan Eich at Netscape in just 10 days. Initially named Mocha, it evolved through several names like LiveScript before being called JavaScript. Today, it's a core part of web development, used for both frontend and backend (Node.js).

1.2 Importance of JavaScript

JavaScript is crucial for adding interactivity, dynamic content, and responsive behavior to websites, making them more engaging and user-friendly. It powers most modern web applications.

1.3 Features of JavaScript

JavaScript is versatile, supporting event-driven programming, dynamic typing, and asynchronous operations. These features enable interactive web experiences and real-time applications, making it the backbone of modern web development.

2. Variables and Data Types in JavaScript

In JavaScript, variables are used to store data, which can change during program execution. You can declare variables using var, let, or const.

2.1 Declaring Variables

A variable is like a container for storing data. To declare a variable, we use one of the keywords: var, let, or const.

2.2 Rules for Naming Variables

Variable names must start with a letter, $, or _, followed by letters, digits, or these symbols. JavaScript reserved words cannot be used as variable names, and variable names are case-sensitive.

2.3 Data Types in JavaScript

JavaScript has two categories of data types: primitive and non-primitive (reference) data types. The primitive data types include null, number, string, boolean, undefined, bigint, and symbol.

2.4 Object Data Types

Objects are a type of reference data type. They can store collections of key-value pairs. Examples of objects in JavaScript include arrays, functions, and user-defined objects.

2.5 Hoisting

Hoisting in JavaScript is the process where variable and function declarations are moved to the top of their scope during the compile phase, allowing variables to be used before they are declared.
